#  Notes App

A lightweight notes-taking app using rich-text editing and localStorage.

##  Features
- Create multiple notes
- Rich text support
- Auto save notes
- Delete notes
- Very simple UI

##  Tech Stack
- HTML
- CSS
- JavaScript
